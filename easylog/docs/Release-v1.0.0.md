# LogPilot - AAOS Log Viewer v1.0.0 Release Notes

**릴리즈 날짜**: 2026-04-23  
**배포 파일**: `LogPilot-AAOS-Log-Viewer-v1.0.0-win-x64.7z`  
**플랫폼**: Windows 10+ (x64), self-contained (.NET 8 포함)

---

## 🚀 v1.0.0 — 정식 릴리즈

LogPilot v1.0.0은 베타 기간(v0.0.1 ~ v0.0.5) 동안 수집된 피드백과 최적화를 반영한 **첫 정식 릴리즈**입니다.

---

## 주요 신규 기능 (v0.0.5 → v1.0.0)

### 1. About 다이얼로그
- ℹ About 버튼 → 버전, 저작권, 버그/개선 문의 메일 표시
- 상단 좌측 버전 배지 제거 → 타이틀바와 About으로 통합

### 2. 다중 파일 타임스탬프 자동 정렬
- 여러 로그 파일을 동시에 열면 **타임스탬프 기준으로 자동 정렬** 후 라인 인덱스 부여
- 파일 로드 순서와 무관하게 시간순 보장

### 3. 대용량 파일 로드 진행률 + 취소
- 파일 로드 중 실시간 진행률 표시 (읽기 → 정렬 → 필터 → 렌더링 단계별)
- **Cancel 버튼**으로 진행 중 취소 가능
- 로딩 팝업 고정 크기(360px) 유지 — 단계 전환 시 레이아웃 흔들림 없음

### 4. 로그 오픈 시 필터 적용 선택
- 활성 필터가 있을 때 파일 열기 시 간결한 한/영 안내 팝업으로 전체 로드 vs 필터 적용 로드 선택

### 5. 스풀 자동 정리
- 앱 시작 시 24시간 이상 된 임시 스풀 파일 자동 삭제
- `%TEMP%\LogPilot\spool\` 경로

### 6. 앱 아이콘 리뉴얼
- 로켓 + 로그 스트림 컨셉의 새 아이콘 적용
- 실행파일, 타이틀바, 작업표시줄, About 창 모두 반영

### 7. 키보드 네비게이션 개선
- **PageUp/PageDown**: 한 페이지 분량(가상화 모드 대응) 이동
- **DataGrid 클릭 시 포커스 자동 이동**: 화살표/PageUp/Down 즉시 동작
- **ESC**: 검색창/DataGrid 포커스 해제

## v0.0.5에서 포함된 주요 기능 (요약)

- **앱 이름 변경**: ALV → LogPilot
- **검색 히스토리**: 최대 10개, 드롭다운, 키보드 탐색, 영속화
- **필터/검색 로딩 오버레이**: 대용량 로그에서 필터/검색 진행 중 사용자 안내
- **메모리 최적화**: Tag 인터닝, LogRowModel 경량화, Snapshot 복사 제거
- **압축 파일 로드 개선**: 다중 zip 내 로그 로드 실패 해결

## 전체 기능 목록

- ADB 실시간 로그 수집 (최근 30초 백필)
- 파일 로드: `.log`, `.logcat`, `.txt`, `.zip`, `.7z` (다중 파일, 드래그&드롭)
- 통합 검색: OR(`|`) / AND(`&`) 다중 검색, 하이라이트
- 필터: Tag/PID/Message/Level include/exclude, 색상 하이라이트
- 필터 세트: JSON 저장/불러오기
- 로그 내보내기: 50MB 분할 7z
- 한글 지원: UTF-8/CP949/EUC-KR 자동 감지
- UI 설정: 폰트 종류/크기, 컬럼 너비/순서 영속화
- Preview 패널: 선택 로그 상세 표시, 텍스트 복사
- Ctrl+C 복사: 행 전체/메시지만/텍스트 선택

## 사전 요구 사항

| 항목 | 용도 | 필수 여부 |
|------|------|-----------|
| Windows 10 이상 (x64) | 앱 실행 | 필수 |
