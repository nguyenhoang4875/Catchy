## 변경 내역

### 1. OFF 상태 필터 편집 시 불필요한 로딩 제거
- 비활성(OFF) 상태 필터를 팝업에서 수정할 때 로그 재적용(로딩 오버레이)이 발생하지 않도록 개선
- ON/OFF 토글 자체는 기존과 동일하게 즉시 반영
- 필터를 자주 편집하는 사용자의 체감 사용성 대폭 향상

### 2. 검색 연타 시 에러(크래시) 수정
- 검색창에서 Enter를 빠르게 반복 입력할 때 발생하던 에러 해결
- CancellationToken 기반으로 이전 검색을 안전하게 취소하고 마지막 검색만 반영
- IsBusy 상태 정합성 보장 — 취소된 검색이 로딩 오버레이를 잘못 해제하지 않음
